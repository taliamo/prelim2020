# Expectation values and means

Expectation value of a general variable A:
$<A> = \frac{\partial ln Z}{\partial P_A}$, where P_A is the factor on A_i and Z is some partition function, dependent on the characteristics of your system (e.g. is it isobaric?)

Fluctuations: $<A^2>-<A^2> = \frac{\partial ^2 ln Z}{\partial P_A^2}=\frac{\partial<A>}{\partial P_A}$

e.g. Prelim Long 2016: for constant T and P use partition function for isobaric-isothermal systems, where A=V and $P_A=-\beta P$, $Z_j=\sum_i e^{-\beta(E_i +pV_i)}$, where $\beta=1/k_BT$

$Z=1/N! (Z_j)^N$.
